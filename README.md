# Logistic Regression Complete Implementation

This repository demonstrates a complete implementation of Logistic Regression using Scikit-Learn, covering binary classification, multiclass classification, hyperparameter tuning, performance evaluation, and imbalanced dataset handling.

##  Topics Covered

### 1. Binary Classification
- Logistic Regression for binary classification problems
- Model training and prediction
- Decision boundary understanding

### 2. Multiclass Classification
- Multiclass Logistic Regression
- One-vs-Rest (OvR) Classification

### 3. Model Evaluation
- Confusion Matrix
- Accuracy Score
- Precision Score
- Recall Score
- F1 Score
- Classification Report

### 4. Hyperparameter Tuning
- GridSearchCV
- RandomizedSearchCV
- Parameter optimization

### 5. Imbalanced Dataset Handling
- Synthetic imbalanced dataset generation
- Class Weight Adjustment
- Performance comparison

### 6. ROC Curve & ROC-AUC
- ROC Curve Visualization
- False Positive Rate (FPR)
- True Positive Rate (TPR)
- ROC-AUC Score Analysis
- Comparison with Dummy Classifier
